Connect Database

Set up Supabase for real-time shared bookings across all employees. Free tier is plenty — setup takes about 3 minutes.

Step 1 — Create a Supabase project

Go to supabase.com, sign up free, click New Project. Choose a name like digidentity-rooms and wait ~60 seconds.

Step 2 — Run this SQL in the SQL Editor

In Supabase: SQL Editor → New Query, paste and run:

CREATE TABLE reservations ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, room_id TEXT NOT NULL, room_name TEXT NOT NULL, room_icon TEXT DEFAULT '🏢', name TEXT NOT NULL, date DATE NOT NULL, slot TEXT NOT NULL, purpose TEXT DEFAULT 'Meeting', created_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE reservations ENABLE ROW LEVEL SECURITY; CREATE POLICY "public_read" ON reservations FOR SELECT USING (true); CREATE POLICY "public_insert" ON reservations FOR INSERT WITH CHECK (true); CREATE POLICY "public_delete" ON reservations FOR DELETE USING (true);
Step 3 — Paste your credentials below

Go to Settings → API in Supabase. Copy the Project URL and anon public key.

Settings → API → Project URL
Settings → API → Project API Keys → anon public
